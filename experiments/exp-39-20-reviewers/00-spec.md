## CRM for Freelancers - MVP

### **3 Personas**
1. **Freelancer** - Primary user managing clients and projects
2. **Client** - Receives invoices and communications
3. **Admin** - System manager (if needed)

### **MVP Features**
- Client database with contact info
- Activity timeline/history
- Invoice creation and tracking
- Address book integration
- Basic reporting

### **5 Screen Wireframes**

**1. Dashboard**
- Quick stats (active clients, pending invoices)
- Recent activity feed
- Upcoming deadlines
- Quick add buttons

**2. Client Profile**
- Contact details (name, email, phone, address)
- Client notes and history
- Project timeline
- Communication log

**3. Activity Feed**
- Chronological activity log
- Filter by client/project
- Add new activity entries
- Status indicators

**4. Invoices**
- Invoice creation form
- Invoice status tracking
- Payment history
- Templates/library

**5. Search**
- Client search bar
- Filter by status, date, project
- Quick access to client profiles
- Advanced search options
