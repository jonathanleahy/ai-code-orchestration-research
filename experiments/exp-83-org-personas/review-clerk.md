# Clerk (Data Entry)

**1. Does this CRM serve YOUR needs? (YES/PARTIALLY/NO)**
**PARTIALLY**

The search and client list are decent, but the client profile editing is painfully slow with all these clicky fields. I can't bulk import or use keyboard shortcuts effectively. The invoicing is okay but not optimized for speed.

**2. What's the #1 thing you need that's MISSING?**
**BULK IMPORT WITH TEMPLATE AND AUTO-FILL**

I need to add 50 clients daily. I want to paste a spreadsheet with columns like Name, Email, Phone, Project Type, and have it auto-create the records with default values. No clicking through 20 fields per client.

**3. What feature would you use MOST?**
**ACTIVITY LOG WITH QUICK-ADD SHORTCUTS**

I'd want to hit Ctrl+Shift+A to quickly log calls, emails, or meetings. Tab order should be optimized for rapid entry - Name → Email → Phone → Activity Type → Notes → Save. Everything else is just noise.

**4. What feature is USELESS to you?**
**REPORTS AND ANALYTICS**

I don't care about charts or reports. I don't even care about seeing my activity stats. I just want to get clients in and get paid. All those dashboards are just visual clutter.

**5. Would you recommend this to a colleague at your level? Why/why not?**
**NO - but only because it's too slow**

I'd recommend it to someone who's new to CRM and doesn't have to enter 50 clients daily. But for speed-critical work, it's a nightmare. The keyboard shortcuts are missing, tab order is wrong, and there's no way to batch process.

**6. Rate 1-10 for YOUR use case.**
**3/10**

It's functional but feels like it was designed by someone who's never actually entered 50 client records in a day. The UI is slow and clunky for my workflow. I'd rather use Excel + a simple database than this for my daily grind.
