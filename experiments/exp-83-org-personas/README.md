# Experiment 83: Role-Based Org Personas

## 5 org levels review the same CRM

| Role | Rating | Serves Needs? |
|------|--------|--------------|
| Clerk (Data Entry) | 3/10 | — |
| Customer (Client-Facing) | 3/10 | — |
| Operator (Day-to-Day) | 3/10 | — |
| Manager (Team Lead) | 3/10 | — |
| Owner (Business) | 4/10 | — |

### Average: 3.2/10
### Cost: $0.0000

### Key Finding
Different org levels need completely different things from the same product.
A clerk wants speed, a customer wants self-service, an owner wants analytics.
