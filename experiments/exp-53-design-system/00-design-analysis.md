# Design Analysis

## Each&Other Website Design (eachandother.com)

### Typography
- Logo: Bold serif/slab, ~32px, black
- Nav: Uppercase, wide letter-spacing, sans-serif, ~13px, black
- Section labels: Uppercase, small (~12px), pink/salmon accent
- H1: Very large (~60-80px), bold serif, black. Accent word in pink.
- H2: ~36px, bold, black
- H3: ~24px, bold, black
- Body: ~16-18px, sans-serif, regular weight, dark, line-height ~1.6
- Accent text: Pink for emphasis (customer experience, user needs)

### Colors
- Background: Pure white #ffffff
- Text: Near-black #1a1a1a
- Accent: Pink/salmon #d4727a (used for emphasis, labels, links)
- Neutral: Grey #666666 (secondary text)
- Logo partners: Greyscale filter on brand logos

### Layout
- Max-width ~1200px, centered with generous margins
- Full-bleed white background
- Hero: left-aligned text, ~50% width
- Sections separated by 80-120px padding
- Logo bar: horizontal, evenly spaced, greyscale
- Services: vertical list with arrow links
- Case studies: image left, text right (alternating)
- Footer: 4-column grid (brand/address, services, about, news)

### Spacing System
- Base: 8px
- xs: 8px, sm: 16px, md: 24px, lg: 48px, xl: 80px, 2xl: 120px
- Section padding: xl-2xl
- Component gap: md-lg

### Components
- Nav: horizontal, logo left, links center-right, lang switcher far right
- Buttons: text links with arrow icon (→), no background, pink on hover
- Cards: minimal, image + text + arrow link
- Service items: text + arrow, full-width, subtle bottom border
- Logo bar: row of greyscale logos with "Who we work with –" label
- CTA section: centered text + "Get in touch" arrow link
- Footer: multi-column with subtle dividers
