# Experiment 53: Design System from eachandother.com

## Process
1. Playwright screenshot of live site
2. DOM analysis for structure, text, layout
3. Visual analysis: colors, typography, spacing
4. AI generates: design tokens, component CSS, example HTML

## Output
- 00-design-analysis.md — visual analysis
- 01-design-tokens.css — CSS custom properties
- 02-components.css — 10 component styles
- 03-example.html — complete page using the design system
